# 10 个 AI Agent 项目方案

## 1. 个人任务 Agent

用自然语言记录任务、调整优先级、生成今日计划。

Agent 能力：

- 从自然语言提取任务
- 创建、更新、查询任务
- 生成日计划
- 总结未完成事项

难度：低到中

## 2. PDF 知识库 Agent

上传文档后，可以基于文档问答，并给出引用来源。

Agent 能力：

- 文档检索
- 基于资料回答
- 引用来源
- 处理追问

难度：中

## 3. 简历与岗位匹配 Agent

对比简历和 JD，输出匹配度、差距、修改建议，并生成新版 bullet points。

Agent 能力：

- 结构化信息抽取
- 评分规则
- 修改建议
- 报告生成

难度：中

## 4. 代码审查 Agent

分析小型仓库或 diff，找问题、建议测试、输出 review summary。

Agent 能力：

- 文件读取工具
- 静态检查
- 问题分级
- 测试建议

难度：中到高

## 5. 网页研究 Agent

输入主题后，Agent 搜索网页、提取关键信息、去重整理，并生成带来源的研究报告。

Agent 能力：

- 搜索
- 页面提取
- 笔记保存
- 报告生成
- 来源追踪

难度：中到高

## 6. 个人记账 Agent

用户输入“今天午饭 38，打车 22”，Agent 自动分类、记录、统计，并生成周报。

Agent 能力：

- 交易信息提取
- 数据库工具
- 自动分类
- 报表生成

难度：中

## 7. 会议纪要 Agent

把会议转写文本整理成决策、行动项、负责人、截止时间和跟进消息。

Agent 能力：

- 长文本总结
- 行动项抽取
- 结构化输出
- 任务创建

难度：中

## 8. 学习教练 Agent

创建学习目标、导入资料、生成测验、追踪进度、调整学习计划。

Agent 能力：

- RAG
- 计划生成
- 记忆
- 进度更新
- 测验生成

难度：中到高

## 9. 电商客服 Agent

基于 FAQ、商品数据、订单数据回答用户问题，无法确定时转人工。

Agent 能力：

- FAQ RAG
- 订单查询工具
- 售后政策判断
- 人工交接

难度：高

## 10. 多 Agent 写作工作室

研究员、提纲作者、初稿作者、审稿人、编辑通过明确流程协作完成文章。

Agent 能力：

- 角色化 Agent
- 共享状态
- 审稿循环
- 最终发布步骤

难度：高

## 推荐你的入手顺序

结合你熟悉前端，也能写后端和数据库的情况，我建议顺序是：

```text
1. 个人任务 Agent
2. PDF 知识库 Agent
3. 学习教练 Agent
4. 网页研究 Agent
5. 代码审查 Agent
6. 多 Agent 写作工作室
```

这个顺序的好处是：一开始就有可见产品反馈，同时逐步加入更深的 Agent 架构能力。
